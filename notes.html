Cons

- Does not support themes yet (I am going to make the dark/light mode in similar way like in the desktop client).

- Popup windows may sometimes bug (extension may think they are still open -> fix button in settings).

- Removed dragging of windows.

- Prepared for BETA Battle Arena -> each NPC kills returns you to previous page.

- Removed some things from previous version -> custom sounds, battle UI, steps UI.

pros

+ Supports opening the game and chat in popup windows (if you close chrome and use the extension and later close the windows of extension, it does not override history last opened tabs).

+ Simpler style of menu ( + some new icons).

+ More space for game frame.

+ Reworked the way logging in is done.

+ Left click on player's name in chat opens menu with quick options: attack, view profile, send item, send gold, spy (popup chat is synchronized with the game tab).

Fixed

+ Quests now show completion message.

+ NPC attack bug.

+ + Wiki pages open in mobile view so they are actually usable at smaller resolutions.
